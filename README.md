# Project Overview
This project analyzes bank loan approval data using Microsoft Excel.
It includes KPI metrics, pivot tables, and an interactive dashboard.

# Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers

# Key Insights
- Approval rate by credit band
- High-risk loans show higher rejection rates
- Income band strongly affects loan approval

# Dataset
- Rows: 150,000+
- Columns: Loan type, income, credit score, duration, approval status
